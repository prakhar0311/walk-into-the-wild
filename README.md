Walk Into The Wild is a full-stack wildlife tourism web application that allows users to explore wildlife destinations, create user accounts, and book safari experiences across India. The project simulates a real-world booking platform with authentication, order management, and an admin dashboard for managing content.


Features
User Features

User registration and login with session-based authentication

Explore wildlife destinations and safari packages

Book safaris using a cart and checkout workflow

View user profile and booking history

Admin Features

Admin dashboard for managing wildlife and safari listings

Create, update, and delete wildlife and safari content

Secure image upload and management

View and manage user bookings and orders
